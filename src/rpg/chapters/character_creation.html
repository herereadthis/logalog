{{ chapter_number }}.2: {{ chapters.character_creation.sub_chapters.backstory_mistakes.title }}

Players need a catalyst

There needs to be something that forces the hero to change by making it impossible to return to their previous life. What event has forced them into a life of adventuring? It doesn't have to be tragic, it could be some fortunate event that allows them to break some hindrance.

Don't ignore world-building and verisimilitude

You can't be an elf if the world doesn't have elves. Hence, character creation shouldn't be the first thing. A character that comes out of the environment is good. Don't make the character incidental to the world.

Players should have a goal in mind

Let the catylst help create the goal, but the goal is just the means to an end. For example, if the goal is to acquire some thing, and the thing gets destroyed, they must still go on.

Backstory does not need to be too long or too detailed

An excessive backstory prevents the character's arc from developing. It makes it harder for the GM to improvise things without conflicting with the backstory.

The backstory and the character's level must be aligned

A beginner-level character cannot be a powerful wizard.
